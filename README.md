# Drone Flight Log

A simple command-line tool to log and review drone flights. Built as a Python practice project.

## What it does
- Logs flights with date, location, duration, purpose, and notes
- Stores everything in a local flights.csv file
- Shows a summary view with total flights and total flight time

## How to run it

python3 flight_log.py

Then choose from the menu:
1. Log a new flight
2. View all flights
3. Quit

## Why I built this
Practice project for learning Python and Git — also genuinely useful for tracking flight hours toward FAA Part 107 currency.
